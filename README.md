# AI Chatbot Evaluation Portfolio

This repository contains structured evaluation case studies analyzing the performance of large language models (LLMs) such as ChatGPT, Claude, and Gemini.

The purpose of this portfolio is to demonstrate practical experience in evaluating AI chatbot behavior across multi-turn conversations, with a focus on:

- context retention
- logical consistency
- instruction following
- response relevance
- multi-turn conversation stability

## Evaluation Method

Each case study follows a structured format:

1. Test Goal  
2. Conversation  
3. Issue Identified  
4. Analysis
